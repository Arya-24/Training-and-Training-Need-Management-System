# Training & Training Need Management System

A full-stack web application developed as part of an internship assessment to manage employees, departments, training courses, and competency mapping. The system helps organizations organize employee information, manage training programs, identify skill gaps, and monitor training requirements through interactive dashboards.

---

## Features

### Employee Management
- View employee list
- Add new employees
- Edit employee details
- Delete employees
- Search employees
- Export employee data (CSV)

### Department Management
- View department list
- Add new departments
- Edit department details
- Delete departments
- Search departments
- Department Dashboard with statistics and charts

### Training Management
- View training courses
- Create new training courses
- Edit existing courses
- Manage course details
- Delete training courses
- Training status and completion tracking

### Competency Mapping (Training Need Identification)
- Employee competency matrix
- Department and employee filtering
- Skill gap analysis
- Training requirement identification
- Add, edit, and delete competency records

### Dashboards
- Admin Dashboard
- Department Dashboard
- Interactive charts and statistics
- Employee and training insights
